# 频率响应

## 1. 一句话定义
[[频率响应]]描述电路或系统输出随输入频率变化的幅值和相位规律。

## 2. 直观理解
扫频时看输出如何变大、变小、滞后或超前。

## 3. 公式 / 表达
H(jω)=输出相量/输入相量。

## 4. 使用条件
- 能用在什么场景：当前章节中需要识别、计算或判断 [[频率响应]] 的题目。
- 不能用在什么场景：题目条件不满足模型、方向、分布或理想化假设时不能硬套。
- 使用前要检查：对象是什么、方向或条件是什么、要和哪些节点一起使用。

## 5. 典型题型 / 应用场景
### 题型一
- 识别特征：题干直接出现 频率响应 或它的等价描述。
- 解题思路：先写定义，再连接 [[网络函数]]、[[幅频特性]]、[[相频特性]]。
- 常见坑点：只背名字，不检查适用条件。

### 题型二
- 识别特征：题目不直接点名，但需要用它和相邻知识点建立关系。
- 解题思路：从已知量出发，判断应链接到 [[RLC串联谐振]]、[[带宽]]、[[品质因数Q]]。
- 常见坑点：漏掉符号、方向、分布对象或理想化条件。

## 6. 我的易错点
- 只看幅值不看相位。
- 不把频率响应和网络函数联系起来。
- 读带宽时没有看半功率点。
