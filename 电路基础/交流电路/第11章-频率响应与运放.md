# 第11章-频率响应与运放

## 4. 本章易错点
- 谐振时电感、电容电压不是 0，而是在总电压中抵消。
- [[品质因数Q]] 越大，[[带宽]]越窄。
- [[虚短]]不是输入端真的短路，[[虚断]]不是电路断开。

## 5. 推荐学习路径
1. [[相量]] -> [[阻抗]] -> [[导纳]]
2. [[网络函数]] -> [[频率响应]] -> [[带宽]]
3. [[RLC串联谐振]] -> [[品质因数Q]]
4. [[理想运放]] -> [[虚短]] / [[虚断]] -> [[节点电压法]]
