# 随机过程学习导航

## 1. 本章主线
[[随机过程]]是一族按时间或指标排列的随机变量。做题时先固定时刻，再把问题还原为普通随机变量或随机向量。

## 4. 本章易错点
- 把[[随机过程]]当成确定函数。
- 把 $\omega$ 误认为角速度。
- 把[[有限维分布]]误写成多个[[一维分布函数族|一维分布]]的乘积。
- 忘记先固定时刻再计算概率。
