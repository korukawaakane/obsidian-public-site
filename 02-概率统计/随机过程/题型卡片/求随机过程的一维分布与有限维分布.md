
# 求随机过程的一维分布与有限维分布

## 1. 题型特征
题目给出一个[[02-概率统计/随机过程/随机过程|随机过程]] $X(t)$ 或 $X_n$，要求求[[02-概率统计/随机过程/知识点卡片/一维分布函数族|一维分布函数族]]、密度函数族或[[02-概率统计/随机过程/知识点卡片/有限维分布|有限维分布]]。

## 2. 核心思想
先固定时刻，再把随机过程还原成普通随机变量或随机向量。

## 3. 标准步骤
1. 看随机源是什么，例如随机变量 $S$、事件 $H$、泊松点数 $N(t)$ 等。
2. 固定题目要求的时刻 $t$ 或 $t_1,\dots,t_n$。
3. 把 $X(t)$ 或 $[X(t_1),\dots,X(t_n)]$ 代出来。
4. 根据类型计算概率：
   - 连续变换：用分布函数法，再求导。
   - 有限条样本轨道：列轨道表，满足条件的概率相加。
   - 计数过程控制：先求计数变量分布，再按偶数、奇数或指定事件求和。
5. 最后写成分布函数、密度函数族或联合分布函数。

## 4. 口诀
一维看一张截图，$n$ 维看同一条轨道的 $n$ 张截图；先代时刻，再算概率；不能默认把多个一维分布相乘，除非已知独立。

## 5. 易错点
- 不要把[[02-概率统计/随机过程/随机过程|随机过程]]当确定函数。
- 不要忘记分段讨论 $x$。
- 离散型过程没有密度函数，只写分布函数或概率质量。
- 连续变换要注意取值范围和单调性。
