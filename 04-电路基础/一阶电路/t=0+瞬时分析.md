# t=0+瞬时分析

## 核心方法

$t=0^+$ 分析不是求稳态，而是把储能元件的初值保留下来后，在换路后的新拓扑中重新列方程。

## 标准步骤

1. 先分析 $t=0^-$ 稳态。
2. 求出电容电压 $u_C(0^-)$ 或电感电流 $i_L(0^-)$。
3. 用 [[换路定律]] 得到 $u_C(0^+)$ 或 $i_L(0^+)$。
4. 在 $t=0^+$ 时，电容用[[电容初值等效|初始电压源等效]]，电感用[[电感初值等效|初始电流源等效]]。
5. 按换路后的新电路列 KCL、KVL 或节点方程。

## 易错点

- $t=0^+$ 不是直流稳态，不能把电容直接开路或把电感直接短路。
- 换路定律只保证 $u_C$ 和 $i_L$ 连续，不保证 $i_C$ 和 $u_L$ 连续。
- 有初始储能时，即使没有独立源，也可能出现瞬时电压或瞬时电流。
