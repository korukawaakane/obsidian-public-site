# 考前急救

## 使用方法

考前优先按下面顺序看：

1. 先看本页的高频知识点。
2. 再看对应 [[题型卡片总索引|题型卡片总索引]]。
3. 最后扫 我的易错点总表。
4. 如果概念混淆，去看 [[我总是搞混的东西|我总是搞混的东西]]。
