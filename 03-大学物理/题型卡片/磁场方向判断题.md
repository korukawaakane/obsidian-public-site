# 磁场方向判断题

## 1. 题型特征
题目要求判断电流、运动电荷或磁场叠加后的方向。

## 2. 核心知识点
- [[03-大学物理/电磁学/磁场方向判断|磁场方向判断]]
- [[03-大学物理/电磁学/长直导线磁场与右手定则|长直导线磁场与右手定则]]
- [[03-大学物理/电磁学/运动点电荷的磁场|运动点电荷的磁场]]
- [[03-大学物理/电磁学/磁场的矢量叠加|磁场的矢量叠加]]

## 3. 标准解法
1. 明确产生磁场的对象：长直导线、圆电流、运动电荷或分布电流。
2. 用右手定则或叉乘方向判断单个贡献。
3. 对多源问题做矢量叠加。
4. 对负电荷，方向相对正电荷反向。

## 4. 易错点
- 长直导线磁场方向是切向，不是径向。
- [[03-大学物理/电磁学/运动点电荷的磁场|运动点电荷的磁场]]中 $\mathbf r$ 从运动电荷指向场点。
- 当 $\mathbf v$ 与 $\mathbf r$ 平行或反向平行时，磁场为 $0$。
