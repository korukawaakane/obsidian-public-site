# 长直导线磁场方向与叠加

## 1. 题型特征
题目给出一根或多根无限长直导线，要求判断某点磁场大小和方向。

## 2. 核心知识点
- [[03-大学物理/电磁学/长直导线磁场与右手定则|长直导线磁场与右手定则]]
- [[03-大学物理/电磁学/磁场的矢量叠加|磁场的矢量叠加]]

## 3. 标准解法
1. 看电流方向。
2. 用右手定则判断磁场绕向。
3. 找目标点到导线的距离 $r$。
4. 套公式 $B=\frac{\mu_0I}{2\pi r}$。
5. 在目标点取圆周切线方向。
6. 多导线时进行矢量叠加。

## 4. 今日理解
- M 点处左右导线产生的磁场可能大小相等、方向相反，因此抵消。
- N 点处左右导线产生的磁场竖直分量可能抵消，水平分量同向相加。
