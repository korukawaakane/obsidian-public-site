
# 电流密度理解易错

## 1. 原始问题
容易把电流密度理解成简单的 I/S 标量。

## 2. 当时卡在哪里
卡在电流密度为什么是矢量，以及它和电流强度的关系。

## 3. 错因分析
概念问题。电流密度描述某点附近电流分布的大小和方向，是局部量。

## 4. 正确理解
电流密度 J 是矢量；总电流 I 是 J 穿过截面的通量，均匀且垂直时才可写成 J=I/S。

2. 再检查适用条件、方向、符号或模型假设。
3. 最后把结果和相关知识点互相验证。
