# 恒定电流

## 1. 本章主线
从[[电荷]]的定向运动定义[[电流]]，用[[电流强度]]描述通过截面的总量，用[[电流密度]]描述局部矢量分布，再用[[电动势]]解释电源维持电流的能力。

## 4. 本章易错点
- 电流方向与电子运动方向不同。
- [[电流密度]]是矢量，不只是电流除以面积。
- [[电动势]]不是普通电压，端电压可能与它不同。

2. [[电流密度]] -> [[欧姆定律]] -> [[电阻率]]
3. [[电动势]] -> [[电功率]] -> [[焦耳定律]]
