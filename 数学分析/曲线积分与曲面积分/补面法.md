# 补面法

## 1. 一句话定义
[[补面法]]是把开曲面补成[[闭曲面]]，再用[[Gauss公式]]计算原曲面积分的方法。

## 2. 直观理解
它像给漏开的曲面补一个盖子，先算整体，再扣掉盖子。

## 3. 公式 / 表达
原曲面积分 = 闭曲面积分 - 补面积分；若原方向与闭曲面外法线相反，结果取负。
