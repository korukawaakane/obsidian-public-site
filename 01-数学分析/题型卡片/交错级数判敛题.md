# 交错级数判敛题

## 1. 题型特征
通项含 $(-1)^n$ 或 $(-1)^{n-1}$，要求判断级数敛散性。

## 2. 核心知识点
- [[01-数学分析/无穷级数/交错级数|交错级数]]
- [[01-数学分析/无穷级数/Leibniz判别法|Leibniz判别法]]
- [[01-数学分析/无穷级数/绝对收敛|绝对收敛]]
- [[01-数学分析/无穷级数/条件收敛|条件收敛]]
- [[01-数学分析/无穷级数/级数收敛的必要条件|级数收敛的必要条件]]

## 3. 标准解法
1. 先检查完整通项 $a_n$ 是否趋于 $0$。
2. 写出去符号后的 $b_n\ge0$。
3. 检查 $b_n\to0$。
4. 检查 $b_n$ 是否单调不增。
5. 若满足[[01-数学分析/无穷级数/Leibniz判别法|Leibniz判别法]]，再判断是否[[01-数学分析/无穷级数/绝对收敛|绝对收敛]]。

## 4. 易错点
- 不能只看到正负交替就判收敛。
- $b_n\not\to0$ 时直接发散。
- 条件收敛和绝对收敛要分开判断。
