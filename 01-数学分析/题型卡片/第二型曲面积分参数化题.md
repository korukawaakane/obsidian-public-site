# 第二型曲面积分参数化题

## 1. 题型特征
给出有向曲面和第二型曲面积分，要求通过参数化计算。

## 2. 核心知识点
- [[第二型曲面积分]]
- [[曲面定向]]
- [[微分形式曲面积分]]
- [[柱面参数化]]

## 3. 标准解法
1. 写出参数化 $\mathbf r(u,v)$。
2. 用 $\mathbf r_u\times\mathbf r_v$ 检查方向。
3. 若方向相反，交换参数顺序或取负。
4. 拉回 $dy\wedge dz$、$dz\wedge dx$、$dx\wedge dy$。
5. 化为二重积分。

## 4. 易错点
- 参数顺序会改变定向。
- 外侧、内侧、上侧、下侧必须和叉乘方向对应。
- $\sqrt{1-\cos^2\theta}=|\sin\theta|$。
