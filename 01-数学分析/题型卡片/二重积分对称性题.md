# 二重积分对称性题

## 1. 题型特征
积分区域具有关于 $x,y$、坐标轴或原点的对称性，被积函数含未知函数但在变量交换下互补。

## 2. 核心知识点
- [[01-数学分析/曲线积分与曲面积分/二重积分对称性|二重积分对称性]]
- [[01-数学分析/曲线积分与曲面积分/积分区域对称性|积分区域对称性]]
- [[01-数学分析/曲线积分与曲面积分/变量交换法|变量交换法]]

## 3. 标准解法
1. 写出积分区域 $D$。
2. 检查 $D$ 在变量交换或对称变换下是否不变。
3. 设出互补积分 $I,J$。
4. 用对称性得 $I=J$。
5. 用 $I+J$ 化成区域面积或简单积分。

## 4. 易错点
- 不需要知道未知函数的具体形式。
- 对称性必须同时作用于区域和被积结构。
