# 分布电流的微元积分法

## 是什么
当电流分布在宽板、面、曲面或线圈上时，要把整体电流拆成微元 $dI$ 或匝数微元 $dN$，先求微元磁场 $dB$，再积分求总磁场。

## 怎么用
1. 判断电流分布。
2. 取电流微元 $dI$。
3. 写微元磁场 $dB$。
4. 判断方向或分量。
5. 利用对称性简化。
6. 积分。

## 易混点
- 不能把宽板、半圆柱面、半球面线圈直接当成一根无限长直导线。
- 不同微元到目标点距离或方向可能不同。
- 对称性分析要明确哪些分量抵消、哪些分量相加。
